Build the Milestone 0 foundation for this project.

Goal:

Set up a clean local development foundation for a modular photo organizer app.

Requirements:

1\. Create a FastAPI backend skeleton in the backend folder.

2\. Add a simple /health endpoint that returns a JSON status response.

3\. Set up a PostgreSQL connection structure using SQLAlchemy.

4\. Set up Redis connection scaffolding for future RQ jobs.

5\. Create a clean Python requirements.txt for the backend.

6\. Create a Next.js + TypeScript frontend skeleton in the frontend folder.

7\. Add a simple frontend home page that confirms the app is running.

8\. Add a top-level README with setup and run instructions.

9\. Keep the code modular and minimal.

10\. Do not implement ingestion or AI features yet.

Important:

\- Keep the code beginner-friendly and easy to test.

\- Use clear filenames and structure.

\- Explain where each file should go.

\- Include commands needed to run each part locally.
