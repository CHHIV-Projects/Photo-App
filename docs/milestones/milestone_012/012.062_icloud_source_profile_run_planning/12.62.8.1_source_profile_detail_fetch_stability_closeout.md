## 2. Source Profile Detail Fetch Stability Findings

- Reproduced: yes
- Error observed:
  - sqlalchemy.exc.OperationalError: (psycopg2.errors.DeadlockDetected)
  - SQL involved in failure path: ALTER TABLE ingestion_sources ALTER COLUMN profile_status SET NOT NULL
  - UI symptom: Failed to fetch in Source Profile Details drawer (often surfaced as CORS-style net::ERR_FAILED after backend 500)
- Schema ensure in detail path: yes (before fix), no (after fix)
- Services called during detail fetch:
  - Endpoint handler: app.api.admin.get_source_profile
  - Direct service: app.services.admin.source_intake_service.get_source_profile_detail
  - Readiness path: app.api.admin.get_source_profile_icloud_readiness -> app.services.admin.icloud_readiness_service.get_icloud_source_readiness -> get_source_profile_detail
- Root cause / best-supported explanation:
  - A schema ensure path that executes DDL (ALTER TABLE ... SET NOT NULL) was reachable during read-time source detail/readiness requests, allowing concurrent UI traffic to trigger lock contention/deadlocks.
- Fix applied:
  - Removed ensure_ingestion_context_schema(...) from these read-oriented methods in source_intake_service:
    - get_source_profile_detail
    - verify_source_profile_path
    - create_source_profile_staging_folder
  - Kept schema ensure behavior in startup/write paths.
- Frontend de-racing added: yes
  - Added request sequence guard in IngestionView detail loader so stale in-flight responses do not overwrite newer drawer state.
- Tests added/run:
  - Added read-path regression tests ensuring schema ensure is not called in read-oriented source profile methods.
  - Added repeated-call API test: 10 sequential calls to /api/admin/source-profiles/7 return 200.
  - Ran backend test files:
    - tests/test_icloud_path_service.py -> passed
    - tests/test_admin_source_profiles_api.py -> passed
  - Ran frontend build -> passed
- Manual rapid-open validation:
  - Against frontend 3002 + patched backend 8002:
    - iCloud profile rapid detail open/close: 12/12 opened, 0 failed fetch
    - local profile rapid detail open/close: 12/12 opened, 0 failed fetch
