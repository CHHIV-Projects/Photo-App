## Safety-Critical Rules

This reconnaissance is for a future destructive local-staging-file milestone. Preserve these rules exactly:

- Cleanup must never delete iCloud cloud-library data.

- Cleanup must never call any cloud delete API.

- Cleanup must never delete Vault files.

- Cleanup must never delete Drop Zone files.

- Cleanup must never delete DB records.

- Cleanup must never delete provenance records.

- Cleanup must never delete Source Profile records.

- Cleanup must never delete source registry / ingestion source records.

- Cleanup must only ever target verified local files inside the selected iCloud Source Profile’s managed staging path.

- Cleanup execution must require explicit operator confirmation.

- Cleanup execution must require a recent successful dry run or must recompute the dry run immediately before execution.

- Cleanup candidates must have positive verification evidence before deletion.

- Cleanup must produce a durable report.

- Cleanup failure must be safe by default.

- Cleanup must not weaken acquisition/intake/cleanup guardrails.

- The app must not store Apple credentials, 2FA codes, session cookies, auth tokens, or secrets.
